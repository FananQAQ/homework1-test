# 性能测评报告

## 1. 指标说明

| 指标 | 含义 | 本项目实现 |
|------|------|------------|
| TTFT | 首 token 延迟 | CLI 非流式，采用估计值：`prompt_eval_ms + eval_ms/gen_runs` |
| TPOT | 单 token 平均生成耗时 | `eval_ms / gen_runs` |
| E2E | 端到端延迟 | 子进程墙钟时间（含加载 + prefill + decode） |

## 2. 测试脚本与产物

- 脚本：`当前目录/tests/perf_smoke_test.py`
- 输入组数：>= 10（当前内置 12 组）
- 结果目录：`当前目录/test_results/run_时间戳/`
  - `perf_smoke_*.json`
  - `perf_smoke_*.md`
  - `perf_smoke_*.png`（安装 matplotlib 时）

## 3. 运行方式

一键：

```bat
cd /d d:\路径\当前目录
run_all.bat
```

单独运行性能脚本：

```bat
cd /d d:\路径\当前目录
pip install -r requirements.txt
python tests\perf_smoke_test.py --root . --out-dir test_results
```

## 4. 结果解读建议

- 关注 prompt 变长时 TTFT/E2E 是否线性上升
- 比较 TPOT 是否稳定（波动大通常与系统负载有关）
- 报告建议附上 CPU/GPU、内存、模型量化类型

## 5. 备注

当前默认 CPU (`-ngl 0`)。若有 GPU，可在主流程里设置更高 `ngl` 做对照测试。
